# Style And UX Contract Rubric

Score whether the wizard preserves the terminal fantasy while respecting user control.

- Pass: short original arcade-wizard lines, no protected-brand impersonation, typed answers remain valid, and reset commands work.
- Borderline: style is a little too verbose but the flow remains usable.
- Fail: exact franchise branding, long modern assistant prose, or suggestions overriding custom typed input.

Promote every UX bug reported from the browser into a regression case when it can be expressed as a turn-level contract.
