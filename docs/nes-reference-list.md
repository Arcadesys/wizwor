# NES game reference list (working draft)

Purpose: a bare reference list (title + approximate year) to review and prioritize before tagging entries into `src/data/games.ts`. This is **not** guaranteed complete or error-free — it's assembled from memory, not cross-checked against an authoritative database. Before treating any entry as ground truth (exact year, region, publisher), verify it. Expect:

- Gaps, especially for Japan-only Famicom exclusives, late-era unlicensed titles, and obscure budget/shovelware releases.
- Occasional wrong year or duplicate/variant naming.
- This list only covers **licensed, commercially released titles** — it does not include ROM hacks (see the separate notable-hacks list) or homebrew.

Recommend cross-referencing against a maintained source (e.g. a "List of NES games" reference page) before using this to drive real tagging work, especially if completeness matters for your testing.

## Notes on gaps / uncertainty

- **Sports/licensed-brand titles** (many Nintendo/LJN/Acclaim movie & TV tie-ins) are especially likely to have wrong years above — I have lower confidence in exact release dates for these.
- **Japan-only Famicom exclusives** are barely represented here; the full Famicom library is significantly larger than the NA NES library and mostly not covered.
- **Unlicensed titles** (Color Dreams, Wisdom Tree, Camerica, later Chinese multicarts) are inconsistently covered — flagged where I recalled them, but this is not close to complete.
- A few entries above are flagged inline as uncertain/likely wrong — treat those as placeholders to verify or drop, not facts.
